02 Typography

제목은 강하게, 설명은 차분하게

출판사 브랜드답게 정보 신뢰도와 문장 가독성을 우선합니다. 과한 장식 서체보다 명확한 산세리프를 기본으로 하고, 명예의 전당이나 인터뷰 라벨에만 세리프 포인트를 씁니다.

Bookswork

Display typography uses tight tracking and strong hierarchy. Korean UI copy stays readable at small sizes.

원고 검토부터 유통까지 필요한 서비스를 연결합니다.

처음 책을 내는 저자도 상담, 견적, 제작, 유통 단계를 쉽게 이해하도록 문장 길이를 짧게 유지합니다.

Caption / 12px / 1.56

필수 입력, 도움말, 상태 설명은 12px 이상으로 유지하고 대비를 낮추되 흐려 보이지 않게 합니다.

09 Feedback / Overlay

상태, 탭, 아코디언, 모달

복잡한 정보를 한 화면에 모두 펼치지 않고, 탭과 아코디언으로 분리합니다. 중요한 확인은 모달, 짧은 결과는 토스트로 처리합니다.

원고에서 책까지

교정, 편집, 디자인, 인쇄, 서점 유통을 단계별로 선택할 수 있습니다.

조건 기반 견적

쪽수, 판형, 부수, 표지 사양에 따라 비용이 달라집니다.

온오프라인 유통

온라인 서점, 오프라인 서점, 저자 직접 판매를 구분해 안내합니다.

원고가 완성되지 않아도 상담 가능한가요?
가능합니다. 현재 완성도, 목차, 샘플 원고를 기준으로 필요한 편집 범위와 출간 방식을 안내합니다.
표지 디자인만 별도로 의뢰할 수 있나요?
가능한 서비스와 불가능한 범위를 명확히 안내하는 FAQ 패턴입니다. 답변은 2-4문장으로 제한합니다.
출간 이후 유통 현황을 확인할 수 있나요?
저자 도구 또는 상담 채널을 통해 판매 및 유통 관련 안내를 제공하는 구조로 확장할 수 있습니다.

11 Motion System

차분하지만 반응이 느껴지는 애니메이션

출판사 브랜드이므로 과한 이펙트보다 입력 피드백, 카드 hover, 폼 제출, 모달 전환에 집중합니다. 모든 모션은 reduced-motion을 지원합니다.

Fade Up

섹션과 카드 진입 시 16px 이하 이동.

animation: fadeUp 260ms cubic-bezier(.16,1,.3,1);

Card Lift

도서 카드와 서비스 카드 hover에 적용.

transform: translateY(-4px);
box-shadow: var(--shadow-md);

Skeleton Shimmer

도서 리스트, 검색 결과 로딩 상태.

background-size: 240% 100%;
animation: shimmer 1.35s linear infinite;

Soft Pulse

새 알림, 현재 단계, 상담 접수 완료.

box-shadow: 0 0 0 16px rgba(225,38,47,0);

Drawer Slide

모바일 상담 메뉴, 빠른 신청 패널.

animation: drawerIn 260ms cubic-bezier(.16,1,.3,1);

Accordion Expand

FAQ, 출판 절차 설명 확장.

details[open] .answer { animation: accordionOpen 260ms; }

13 Responsive / Accessibility

반응형과 접근성 운영 규칙

저자 연령대가 넓은 출판사 사이트는 모바일 CTA, 충분한 글자 크기, 명확한 포커스가 중요합니다.

Responsive behavior

Do / Don't

Do
  • CTA는 화면당 하나를 primary로 유지
  • 상담 폼은 단계와 도움말을 함께 표시
  • 도서 카드에는 실제 정보 우선 표시
  • 전화번호, 카톡, 온라인 상담을 같은 액션군에 배치
Don't
  • 빨간색을 모든 버튼과 텍스트에 남용
  • 초기 화면에 너무 많은 폼 항목을 배치
